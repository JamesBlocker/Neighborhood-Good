# Neighborhood Good 
A community portal to connect neighbors with the local events that are important and fun to them

### Etch App
<https://neighborhood-good.herokuapp.com/>

# Technologies Used

1. NodeJS, React, Javascript, JSON, JQuery, HTML, CSS, Bootstrap, API, NPM Packages (Axios, Bcrypt-nodejs, body-parser, connect-flash, connect-mongodb-session, cookie-parser, dotenv, ejs, express, express-ejs-layouts, express-session, method-override, mongoose, morgan, passport, passport-local, watson-developer-cloud)

# User Stories

1. As a user, I want to be able to create an account, log in and log out of my account. 
2. As a user, I want to be able to create a community event posting that others can view.
3. As a user, I want to be able to edit posts I've made.
4. As a user, I want to be able to delete posts that I have created.
5. As a user, I want to be able to see a feed of everyones posts.

# Approach Taken

1. I wanted to create a project that was useful and easy to use.
2. I wanted to create an app that would be of benefit to a community.  
3. I wanted to created an app with little/no help to test my skills.
